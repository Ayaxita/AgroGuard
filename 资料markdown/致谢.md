# 致  谢

行文至此，我的本科生涯也即将结束。四年的时光转瞬即逝，在这宝贵的四年，我学到了很多，也收获了很多。

首先，特别感谢我的指导老师崔晶教授。从课题的选取、系统设计、代码实现到最后论文的撰写，崔老师给我提了许多宝贵的意见。崔老师严谨的教学态度和负责任的工作态度让我受益匪浅。

其次，我要感谢我的父母。感谢我的父母二十多年来对我的栽培，感谢他们这么多年无怨无悔的付出和照顾，是他们对我的支持，让我有了继续前进的勇气。

最后，我要感谢我的朋友们。感谢他们对我学习上的支持，以及生活中的陪伴。他们让大学生活变得更加有色彩，未来我们一起努力，一起进步。

感谢这一路上所有的相遇，感谢每一个帮助我的人，我们一定会变得更好，再次相遇！
