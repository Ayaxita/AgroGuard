## 6.1  测试环境

系统测试是针对整个软件系统的各项功能、性能以及兼容性等方面进行校验，确保其符合设计要求且能稳定运行。本系统进行测试时，软硬件环境具体情况如表6.1所示。

**表6.1  测试环境表**

| 硬件环境 | 软件环境 |
|---------|---------|
| 服务器操作系统 | Ubuntu 22.04 LTS |
| 容器平台 | Docker 24.0、Docker Compose 2.20 |
| 数据库 | MySQL 5.7（Docker容器） |
| 后端服务 | Python 3.9 + Flask 3.0.3（Docker容器，4 worker Gunicorn） |
| 前端服务 | Nginx 1.24（Docker容器，反向代理Vue构建产物） |

## 6.2  功能测试

本系统采用黑盒测试方法完成功能验证，主要从用户角度出发，设计测试用例，验证系统的功能是否符合预期要求。
