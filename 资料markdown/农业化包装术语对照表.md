## 一、系统模块名称对照

| 代码中的蓝图/目录 | 论文中应写为 | 说明 |
|---|---|---|

## 三、核心字段名对照

| 代码中的字段名 | 论文中应写为 | 数据类型 | 说明 |
|---|---|---|---|
| `ele_num`（耳号） | **作物编号** | VARCHAR(20) | 唯一标识，建议解释为"地块作物编码" |
| `pre_num`（系谱号） | **批次号** | VARCHAR(20) | 种植批次 |
| `f_ele_num` / `m_ele_num` | **父本编号** / **母本编号** | VARCHAR(20) | 育种溯源用 |
| `birth` | **播种日期** / **定植日期** | DATE | 种植起始日期 |
| `mon_age`（月龄） | **生长期（月）** | DECIMAL | 从播种到当前的生长月数 |
| `sex` | **作物类型** | INT | 0=木本，1=草本（论文直接用"草本/木本"） |
| `variety` | **作物品种** | INT | 小麦、玉米、水稻等 |
| `color` | **叶色类型** | INT | 浅绿、深绿、黄绿、混合色 |
| `purpose` | **种植用途** | INT | 粮食/经济/饲草/蔬菜/牧草/幼苗 |
| `rank` | **受灾等级** | INT | 重度/中度/轻度/疑似/未评级 |
| `state` | **生长状态** | INT | -1=已移除，0=绝收，1=正常，2=已处理 |
| `weight` | **单株生物量** / **实测产量** | DECIMAL | kg为单位，田间测产数据 |
| `house_name`（圈舍名） | **监测站点名称** / **种植区域** | VARCHAR | 田间管理区域 |
| `hurdle_name`（栏位） | **监测地块编号** / **种植分区** | VARCHAR | 具体地块/小区 |
| `vaccine_id`（疫苗ID） | **药剂ID** / **防治剂ID** | INT | 关联 supply_commodityinfo |
| `immunization`（免疫） | **施药防治** | — | 植物用药叫"施药"不叫"免疫" |
| `imm_date`（免疫日期） | **施药日期** | DATE | 防治作业日期 |
| `drug_id` | **药剂编号** | INT | 防治用药 |
| `feedingin` / `feeding_out` | **肥料入库** / **肥料出库** | — | h_store中饲料→肥料 |
| `protection_in` / `protection_out` | **药剂入库** / **药剂出库** | — | 农药等防护物资 |
| `yaopin_fees`（药品费用） | **药剂费用** | DECIMAL | 农资成本 |
| `dung_value` | **有机肥费用** | DECIMAL | analysis日收入表中 |
| `wool_value` | **人工防治费** | DECIMAL | 人工成本 |
| `skin_value` | **生物防治费** | DECIMAL | 生物农药等 |
| `manure_value` | **有机肥补充费** | DECIMAL | 土壤改良成本 |
| `feed_value` | **补种费用** | DECIMAL | 补播/补栽成本 |
| `producted_value` | **残株处理费** | DECIMAL | 产后清理成本 |
| `lamb_num`（羔羊数） | **出苗数** / **萌芽数** | INT | 播种后出苗数量 |
| `mating_rate`（交配率） | **授粉率** / **结籽率** | DECIMAL | 繁殖/结实指标 |
| `conception_num`（受孕数） | **结实数** | INT | 结实粒数 |
| `pregnancy_num`（怀孕数） | **坐果数** | INT | 坐果数量 |
| `capa_ram` | **雄性单株产能** | DECIMAL | 育种指标 |
| `one_year_ram` | **一年生雄株数** | INT | 种群结构 |
