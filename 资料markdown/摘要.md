# 摘    要

草地作物管理长期依赖纸质档案和人工统计，数据分散、更新慢，病虫害防治信息经常滞后。为此设计并实现了一套Web端草地作物病虫害智能监测系统（以下简称草地病虫害监测系统），把基础档案、田间记录、施药防治、物资仓储、销售结算等业务整合到线上统一处理。

后端采用Python Flask框架，按蓝图拆分用户认证、档案、田间、防治、物资、销售等业务模块，SQLAlchemy操作MySQL，JWT做登录鉴权，APScheduler定时任务每日凌晨自动刷新病虫害防护预警。前端用Vue 3和Element Plus搭建页面，Vue Router管理路由，Pinia管理状态，ECharts渲染图表。系统接入智谱GLM大模型接口，通过Function Calling提供带数据库查询的农业问答助手。项目以Docker Compose编排MySQL、Flask和Nginx服务，实现容器化部署。

功能覆盖草地档案维护、田间场地管理、繁殖培育、施药防治与自动预警、物资仓储、收获销售、资产估值和日收入统计，支持Excel批量导入与二维码追溯。分管理员和普通用户两级权限。实际运行中，该系统把分散的手工流程转到线上，缩短了数据查询和病虫害响应时间，减少了重复录入与统计工作量。

**关键词：** 草地作物管理；病虫害预警；Flask；Vue.js；MySQL
