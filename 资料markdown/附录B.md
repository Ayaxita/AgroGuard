# 附录B  软件使用说明书

## 1.硬件环境

（1）CPU：Intel i5-10400或同等性能处理器及以上；
（2）内存容量：8G及以上，推荐16G；
（3）硬盘容量：100G及以上剩余空间。

## 2.软件环境

（1）操作系统：Ubuntu 22.04 LTS / Windows 11 / macOS 13；
（2）浏览器：Google Chrome 120+ / Mozilla Firefox 121+ / Microsoft Edge 120+；
（3）开发工具：VS Code 1.85+、Navicat Premium 16；
（4）开发环境：Python 3.9、Node.js 18+、npm 9+；
（5）容器平台：Docker 24.0、Docker Compose 2.20；
（6）数据库：MySQL 5.7。

## 3.部署过程

### 方式一：Docker Compose一键部署（推荐生产环境）

（1）确保服务器已安装Docker和Docker Compose；
（2）将项目代码上传至服务器指定目录；
（3）进入项目根目录，执行以下命令构建并启动所有服务：

```bash
docker-compose up -d
```

（4）等待容器启动完成，Docker Compose将自动创建并运行三个容器：MySQL数据库容器（端口5010）、Flask后端服务容器（端口5000）、Nginx前端代理容器（端口5020）；
（5）使用Navicat Premium连接MySQL数据库，导入init-scripts目录下的meadow_test.sql初始化脚本；
（6）在meadow-be目录下创建.env文件，配置ZHIPU_API_KEY等环境变量；
（7）浏览器访问http://服务器IP:5020即可进入系统登录页面。

### 方式二：本地开发环境部署（推荐开发调试）

**后端部署：**
（1）确保本地已安装Python 3.9和MySQL 5.7；
（2）使用Navicat Premium创建数据库meadow，导入meadow_test.sql初始化脚本；
（3）进入meadow-be目录，安装Python依赖包：

```bash
pip install -r requirements.txt
```

（4）修改App/__init__.py中的数据库连接配置，确保与本地MySQL的账号密码一致；
（5）运行Flask应用：

```bash
python app.py
```

后端服务将在http://localhost:5000启动。

**前端部署：**
（1）确保本地已安装Node.js 18+；
（2）进入meadow-fe目录，安装前端依赖包：

```bash
npm install
```

（3）修改.env.development文件中的VITE_API_URL为本地后端地址http://localhost:5000；
（4）启动前端开发服务器：

```bash
npm run dev
```

前端服务将在http://localhost:5020启动。

## 4.访问系统

输入http://localhost:5020即可进入系统登录页面。

（1）进入首页，可以查看系统数据看板和待处理预警数量概览，点击右下角悬浮按钮可打开大模型问答助手。

（2）进入基本信息模块，点击"草地信息"页面可以查看和管理草地作物档案，支持按作物编号、品种、状态等条件筛选，点击"新增"按钮填写作物编号、批次号、品种、播种日期等信息后保存；点击"草地健康状态"页面可以查看和登记田间长势调查数据。

（3）进入批次管理模块，点击"生长周期监测"页面可以查看各批次作物的完整生长历程；点击"阶段切换信息"页面可以登记温室苗向大田移栽前的环境适应数据；点击"采样信息"页面可以记录定期田间采样和检测指标。

（4）进入草地护理模块，点击"草地防护"页面可以查看施药防治记录，系统会自动关联显示作物编号、药剂名称和供应商名称，点击"新增"按钮登记新的施药信息。

（5）进入预警信息模块，点击"病虫害预警"页面可以查看系统自动生成的防护到期提醒列表，即将超期的记录以红色高亮显示。

（6）进入库存管理模块，点击"防护物资库存信息"页面可以查看当前库存总览，余量低于阈值的商品以红色高亮提示需要补货；点击"出库"→"防护物资出库信息"页面可以登记药剂的使用出库记录，系统会自动校验库存余量并扣减。

（7）进入产品收割销售管理模块，点击"草地销售"页面可以登记作物收获后的销售结算信息，选择作物批次后系统自动带出品种和估值，填写销售数量和单价后系统自动计算总价。

（8）进入草地病虫害监测分析模块，点击"草地资产"页面可以查看按品种分组的资产估值列表；点击"数据可视化"页面可以查看资产分布饼图和收入趋势折线图；点击"日收益报表"页面可以查看每日的收入支出明细。
