# 草地作物病虫害智能监测系统 (AgroGuard)

> **重要提示**：本项目是**草地作物病虫害智能监测系统**，而非羊场管理系统。在使用本项目时，请始终将其理解为病虫害监测相关系统。

## 项目简介

AgroGuard 是一套专为草地和农作物病虫害监测而设计的智能化管理系统。该系统集成了田块管理、病虫害预警、作物生长监测、防治物资管理、数据统计分析和智能诊断等功能，为现代农业植保工作提供全面的信息化支撑。

**本项目由羊场管理系统重构而来，但在功能描述上已完全转变为草地作物病虫害监测系统。**

- **框架**: Vue 3.4 + TypeScript

## 功能模块说明

以下是系统功能与病虫害监测系统的对应关系：

| 编号 | 原羊场功能模块 | 映射为病虫害监测系统 | 功能描述 |
|------|--------------|-------------------|---------|
| 01 | 羊只基本信息 | **田块/作物信息管理** | 记录田块位置、面积、作物种类、种植时间等信息 |
| 02 | 圈舍管理 | **草地分区管理** | 管理不同草地/田块区域及其边界 |
| 03 | 繁殖管理 | **病虫害繁殖周期监测** | 记录病虫害的发生周期、繁殖规律 |
| 04 | 健康/免疫管理 | **病虫害预警监测** | 病虫害预警设置、防治计划制定 |
| 05 | 疾病记录 | **病虫害发生记录** | 记录病虫害发生时间、类型、严重程度 |
| 06 | 死亡/淘汰记录 | **病虫害损失记录** | 记录因病虫害导致的作物损失情况 |
| 07 | 饲料供应 | **农药/防治物资管理** | 农药、化肥等防治物资的库存管理 |
| 08 | 销售记录 | **病虫害处理报告** | 记录病虫害处理措施和结果 |
| 09 | 资产统计 | **病虫害损失评估** | 统计病虫害造成的经济损失 |
| 10 | 日/周/月报 | **监测数据分析报表** | 病虫害监测数据分析报表 |
| 11 | 智能问答(ChatBox) | **病虫害智能诊断** | 基于AI的病虫害诊断助手 |

## 核心功能详解

### 1. 田块/作物信息管理
- 田块基本信息（位置、面积、承包人）
- 作物种类与品种记录
- 生长周期跟踪
- 作物照片档案管理

### 2. 草地分区管理
- 草地/田块区域划分
- 区域责任人分配
- 区域面积统计
- 区域状态管理（正常/预警/受灾）

### 3. 病虫害繁殖周期监测
- 病虫害种类库管理
- 繁殖周期记录
- 预测预警功能
- 历史数据分析

### 4. 病虫害预警监测
- 预警阈值设置
- 预警通知管理
- 防治计划制定
- 防治效果跟踪

### 5. 病虫害发生记录
- 病虫害发生时间、地点
- 病虫害类型与等级
- 危害程度评估
- 处置措施记录

### 6. 病虫害损失评估
- 损失面积统计
- 经济损失计算
- 产量损失评估
- 价值核算报表

### 7. 农药/防治物资管理
- 物资入库/出库
- 库存盘点
- 物资使用统计
- 供应商信息管理

### 8. 病虫害智能诊断
- 基于DeepSeek AI的智能问答
- 病虫害图片识别（预留）
- 诊断建议生成
- 知识库查询

## 运行指南


1. **本系统是草地作物病虫害智能监测系统**，所有功能模块均应理解为与病虫害监测相关
