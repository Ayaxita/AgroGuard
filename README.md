# AgroGuard

AgroGuard 是一个智能监测系统的项目代码仓库。

## 项目简介

本项目包含前后端完整代码，致力于提供高效的监测与管理解决方案。

## 技术栈

- 后端：Python
- 前端：Vue.js + TypeScript
- 部署：Docker

## 项目结构

```
.
├── docker-compose.yml    # Docker 编排配置
├── init-scripts/         # 初始化脚本
├── meadow-be/            # 后端服务
└── meadow-fe/            # 前端应用
```

## 快速开始

```bash
# 使用 Docker Compose 启动
docker-compose up -d
```
